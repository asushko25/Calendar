# Calendar 

**Live Demo:** [https://asushko25.github.io/Calendar/](https://asushko25.github.io/Calendar/)

## Stack

* React 19 + TypeScript
* Vite 7
* Tailwind CSS 3
* SCSS (Dart Sass via `sass-embedded`)
* Deployed with `gh-pages`

## Setup

```bash
npm install
npm run dev   # start dev server
```
